# Quick Start Guide - KerbVision IR

## 5-Minute Setup

### 1. Install (30 seconds)
1. Download `KerbVisionIR-v1.0.0.zip`
2. Extract the `GameData` folder to your KSP installation
3. Launch KSP

### 2. First Flight (1 minute)
1. Start any flight (or load a save)
2. Look for the green camera icon ?? in the toolbar
3. Press **Alt + I** to toggle night vision on/off

### 3. Adjust Settings (2 minutes)
1. Click the green camera icon to open settings
2. Try different vision modes (click the mode button to cycle):
   - **Grayscale** - Clean monochrome
   - **Green NV** - Classic military night vision
   - **IR High Contrast** - Thermal imaging style
3. Adjust sliders to your taste:
   - **Brightness** ? overall exposure
   - **Contrast** ? image contrast
   - **Gamma** ? gamma correction
   - **Tint Strength** ? color intensity
   - **Grain** ? realistic noise effect
4. Click **Apply** to save

### 4. Recommended Settings for Beginners

**For Night Flights:**
- Mode: Green NV
- Brightness: 0.65
- Contrast: 0.55
- Gamma: 0.50
- Tint Strength: 0.70
- Grain: Enabled (0.20)
- Bloom: Disabled

**For Space Operations:**
- Mode: Grayscale
- Brightness: 0.50
- Contrast: 0.50
- Gamma: 0.50
- Tint Strength: 0.00
- Grain: Enabled (0.15)
- Bloom: Disabled

### 5. Tips

? **DO:**
- Use during night operations for better visibility
- Adjust gamma if image looks washed out
- Use tint strength to control color intensity
- Keep grain enabled for realism

? **DON'T:**
- Enable bloom if you have performance issues
- Set brightness and gamma too high (washes out detail)
- Forget to press Apply after changing settings

## Troubleshooting

**Can't see the effect?**
- Make sure it's toggled ON (press Alt + I)
- Increase brightness slider
- Check you're in Flight scene

**Performance issues?**
- Disable Bloom effect
- Lower Grain intensity
- Close settings window

**Settings not saving?**
- Click Apply button before closing
- Check file permissions in GameData folder
