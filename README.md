# KerbVision IR

Night Vision mod for Kerbal Space Program with old-school CRT artifacts.

## Features

- **3 Vision Modes:** Monochrome, Green NV, Amber/Warm
- **Adjustable Settings:** Brightness (0-2), Contrast (-1 to 1), Color Tint, Grain (0-4)
- **Processing Quality:** Full, High, Medium, or Low (affects performance)
- **CRT Artifacts** (at high grain): Scanlines, phosphor spots, rolling bars, vignette
- **Audio:** Activation sound (place NVon.wav in Sounds folder)
- **Hotkey:** Alt + ` (backtick)

## Installation

### Via CKAN (Recommended)
1. Open CKAN
2. Search for "KerbVision IR"
3. Click Install
4. Launch KSP

### Manual Installation
1. Download latest release from [GitHub Releases](https://github.com/garyblu71mods/KerbVisionIR/releases)
2. Extract to `GameData/KerbVisionIR/`
3. Launch KSP
4. Press **Alt + `** in flight to toggle

## Usage

- **Alt + `** - Toggle night vision on/off
- **Toolbar button** - Open settings window
- Settings save automatically

## Performance

Processing quality settings (lower = better FPS):
- **Low** (default): ~10% FPS drop - Recommended for most users
- **Medium**: ~20% FPS drop
- **High**: ~35% FPS drop
- **Full**: ~60% FPS drop - Only for powerful PCs

Visual effects:
- Grain 0-2: Basic effects only
- Grain 2-4: Full CRT artifacts (scanlines, spots, rolling bars)

## Requirements

- KSP 1.8+
- .NET Framework 4.8
